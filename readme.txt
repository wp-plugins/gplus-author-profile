== Description ==

Simple plugin to get your author's profile linked in Google. 
 Grants autorship photo in Google SERP. Visit blog.ljasinski.pl for more details.
 All settings are per user, change them in your profile.
 
Beware - plugin turns off default user's bio filtering. Don't use it, if you can't
trust your users not to harm your website!
== Installation ==

Download and copy to plugins dir :)
